# MarkdownCore

Core components and utilities shared across markdown modules.

## Overview

MarkdownCore provides foundational components, hooks, and utilities used by the Markdown and InlineLayout modules.

## Exports

- `Components` - Base React components
- `Enums` - Shared enumeration types (e.g., InlineEngineTag)
- `Utils` - Common utility functions
- `StoryMiddleware` - Storybook middleware for testing
